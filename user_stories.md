# Smart Clinic Management System - User Stories

## Doctor
- As a Doctor, I can log in so that I can access my dashboard.
- As a Doctor, I can view my upcoming appointments.
- As a Doctor, I can see patient records so that I can provide accurate medication.

## Patient
- As a Patient, I can register and log in.
- As a Patient, I can search and book appointments with doctors.
- As a Patient, I can view my upcoming appointments.

## Admin
- As an Admin, I can log in to the admin portal.
- As an Admin, I can add new doctors.
- As an Admin, I can manage users and view reports.
